앱 설치 가능

WackyDocs를 홈 화면에 추가하여 앱처럼 사용하세요!

iOS (Safari)
  1. Safari에서 WackyDocs에 접속
  2. 화면 하단의 공유 버튼 탭
  3. "홈 화면에 추가" 선택
  4. "추가" 버튼으로 설치 완료
Android (Chrome)
  1. Chrome에서 WackyDocs에 접속
  2. 주소창에 "설치" 버튼 확인
  3. "설치" 버튼 탭
  4. "설치" 확인으로 완료
